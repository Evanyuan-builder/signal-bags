§ What Signal is — and isn't

A launch strategist, not a launcher.

Given a launch idea, Signal judges whether the narrative is launch-ready, routes 2–3 comparable paths (each citing a named Bags token as proof), and flags if the draft is already a word-for-word clone of an existing on-chain launch. Every number Signal cites traces back to a concrete Bags launch in its local index — no LLM-hallucinated comps.

Signal is
  • A strategy copilot, grounded in 2,000+ real Bags launches
  • A clone-detector for draft descriptions
  • An honest "I don't know" when no comp exists
Signal isn't
  • A launcher — does not sign the launch transaction
  • A hype copy generator — no tagline, no ticker naming
  • A trading bot — doesn't read prices or signal buys/sells

§ Demo

Three scenarios, one harness.

01 · Strong match

AI-assistant streaming community token

Signal cites 4 comparable tokens in the 0.40–0.51 similarity range and proposes 3 concrete launch routes, each grounded in a named Bags launch.

02 · Honest "I don't know"

Niche AI code-review token

Signal names TURINGMIND as the closest comp but flags it as inactive and refuses to commit to a route. The right answer isn't always "launch" — sometimes it's "wait."

03 · Clone detection

Generic community-token draft

Signal surfaces two word-for-word identical existing launches and calls the draft out as a clone. Refuses to suggest a path — tells the user to reframe first.

§ Architecture

Data-grounded agent, zero-modification harness.

Data. Bags API (/token-launch/feed + /solana/bags/pools) + Helius DAS → SQLite.
Semantic. sentence-transformers/all-MiniLM-L6-v2 on MPS, brute-force cosine.
Agent. MiniMax-M2.7 via EvanCore's OpenAIModelClient.
Harness. EvanCore v2 backends — signal:// for roles, bags:// for tools. Zero core changes.
UI. FastAPI + SSE streaming agent trace to the browser.
